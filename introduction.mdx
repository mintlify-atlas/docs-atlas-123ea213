## What is ACE-Step 1.5?

ACE-Step 1.5 is a highly efficient open-source music foundation model that brings **commercial-grade generation to consumer hardware**. It achieves quality beyond most commercial music models while remaining extremely fast — under 2 seconds per full song on an A100 and under 10 seconds on an RTX 3090.

## Key Features

### Performance
- **Ultra-Fast Generation** — Under 2s per full song on A100, under 10s on RTX 3090
- **Flexible Duration** — Generate from 10 seconds to 10 minutes (600s)
- **Batch Generation** — Create up to 8 songs simultaneously

### Quality & Versatility
- **Commercial-Grade Output** — Quality between Suno v4.5 and Suno v5
- **Rich Style Support** — 1000+ instruments and styles with fine-grained timbre description
- **Multi-Language Lyrics** — Supports 50+ languages

### Advanced Features
- ✅ Reference Audio Input for style guidance
- ✅ Cover Generation from existing audio
- ✅ Repaint & Edit capabilities
- ✅ Track Separation into individual stems
- ✅ Multi-Track Generation
- ✅ Vocal2BGM auto-accompaniment
- ✅ Metadata Control (BPM, key/scale, time signature)
- ✅ LoRA Training for personalized styles

## Architecture Overview

ACE-Step 1.5 features a novel hybrid architecture where the Language Model (LM) functions as an omni-capable planner:

- **Language Model** — Transforms simple queries into comprehensive song blueprints, synthesizing metadata, lyrics, and captions via Chain-of-Thought reasoning
- **Diffusion Transformer (DiT)** — Generates high-quality audio based on the LM's planning
- **Intrinsic Reinforcement Learning** — Alignment achieved through internal mechanisms, eliminating external reward model biases

## System Requirements

<Info>
**Minimum:** Python 3.11-3.12, 4GB VRAM (with INT8 quantization + CPU offload)

**Recommended:** 8GB+ VRAM for optimal performance with Language Model
</Info>

### GPU Compatibility

| Your GPU VRAM | Recommended LM Model | Backend | Notes |
|---------------|---------------------|---------|-------|
| **≤6GB** | None (DiT only) | — | LM disabled by default |
| **6-8GB** | `acestep-5Hz-lm-0.6B` | `pt` | Lightweight LM |
| **8-16GB** | `acestep-5Hz-lm-0.6B` / `1.7B` | `vllm` | 0.6B for 8-12GB, 1.7B for 12-16GB |
| **16-24GB** | `acestep-5Hz-lm-1.7B` | `vllm` | 4B available on 20GB+ |
| **≥24GB** | `acestep-5Hz-lm-4B` | `vllm` | Best quality, no offload needed |
