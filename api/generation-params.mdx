---
title: GenerationParams
description: Configuration dataclass for music generation parameters
---

`GenerationParams` is a dataclass that encapsulates all parameters for music generation, including text inputs, metadata, generation settings, and task-specific options.

## Constructor

```python
from acestep.inference import GenerationParams

params = GenerationParams(
    caption="Epic orchestral music with soaring strings",
    lyrics="[Instrumental]",
    bpm=120,
    duration=30,
    inference_steps=8,
    seed=42
)
```

## Text Input Parameters

<ParamField path="caption" type="str" default="''">
  Short text prompt describing the desired music (main prompt). Maximum 512 characters.
</ParamField>

<ParamField path="lyrics" type="str" default="''">
  Lyrics for the music. Use `"[Instrumental]"` for instrumental songs. Maximum 4096 characters.
</ParamField>

<ParamField path="instrumental" type="bool" default="False">
  If `True`, generate instrumental music regardless of lyrics.
</ParamField>

<ParamField path="global_caption" type="str" default="''">
  Global/song-level caption for SFT-stems lego tasks.
</ParamField>

## Music Metadata

<ParamField path="bpm" type="Optional[int]" default="None">
  BPM (beats per minute), e.g., 120. Set to `None` for automatic estimation. Valid range: 30-300.
</ParamField>

<ParamField path="keyscale" type="str" default="''">
  Musical key (e.g., `"C Major"`, `"Am"`). Leave empty for auto-detection. Format: A-G, with optional #/♭, major/minor.
</ParamField>

<ParamField path="timesignature" type="str" default="''">
  Time signature (`"2"` for 2/4, `"3"` for 3/4, `"4"` for 4/4, `"6"` for 6/8). Leave empty for auto-detection.
</ParamField>

<ParamField path="vocal_language" type="str" default="'unknown'">
  Language code for vocals, e.g., `"en"`, `"zh"`, `"ja"`, or `"unknown"`. See `acestep/constants.py:VALID_LANGUAGES`.
</ParamField>

<ParamField path="duration" type="float" default="-1.0">
  Target audio length in seconds. If `<0` or `None`, model chooses automatically. Valid range: 10-600.
</ParamField>

## Audio Post-Processing

<ParamField path="enable_normalization" type="bool" default="True">
  Whether to apply loudness normalization to the output audio.
</ParamField>

<ParamField path="normalization_db" type="float" default="-1.0">
  Target loudness in dB for normalization (e.g., -1.0 for -1 dBFS peak).
</ParamField>

<ParamField path="fade_in_duration" type="float" default="0.0">
  Fade in duration in seconds. `0` = no fade in.
</ParamField>

<ParamField path="fade_out_duration" type="float" default="0.0">
  Fade out duration in seconds. `0` = no fade out.
</ParamField>

<ParamField path="latent_shift" type="float" default="0.0">
  Additive shift applied to DiT latents before VAE decode. Default `0` = no shift.
</ParamField>

<ParamField path="latent_rescale" type="float" default="1.0">
  Multiplicative rescale applied to DiT latents before VAE decode. Default `1.0` = no rescale.
</ParamField>

## Generation Parameters

<ParamField path="inference_steps" type="int" default="8">
  Number of diffusion steps (e.g., 8 for turbo, 32-100 for base model).
</ParamField>

<ParamField path="guidance_scale" type="float" default="7.0">
  CFG (classifier-free guidance) strength. Higher means following the prompt more strictly. Only supported for non-turbo model.
</ParamField>

<ParamField path="seed" type="int" default="-1">
  Integer seed for reproducibility. `-1` means use random seed each time.
</ParamField>

<ParamField path="infer_method" type="str" default="'ode'">
  Diffusion inference method. Options: `"ode"` or `"sde"`.
</ParamField>

<ParamField path="timesteps" type="Optional[List[float]]" default="None">
  Custom timesteps (parsed from string like `"0.97,0.76,0.615,0.5,0.395,0.28,0.18,0.085,0"`). If provided, overrides `inference_steps` and `shift`.
</ParamField>

## Advanced DiT Parameters

<ParamField path="use_adg" type="bool" default="False">
  Whether to use Adaptive Dual Guidance (only works for base model).
</ParamField>

<ParamField path="cfg_interval_start" type="float" default="0.0">
  Start ratio (0.0-1.0) to apply CFG.
</ParamField>

<ParamField path="cfg_interval_end" type="float" default="1.0">
  End ratio (0.0-1.0) to apply CFG.
</ParamField>

<ParamField path="shift" type="float" default="1.0">
  Timestep shift factor (default 1.0). When `!= 1.0`, applies `t = shift * t / (1 + (shift - 1) * t)` to timesteps.
</ParamField>

## Task-Specific Parameters

<ParamField path="task_type" type="str" default="'text2music'">
  Type of generation task. Options: `"text2music"`, `"cover"`, `"repaint"`, `"lego"`, `"extract"`, `"complete"`.
</ParamField>

<ParamField path="instruction" type="str" default="'Fill the audio semantic mask based on the given conditions:'">
  Optional task instruction prompt. If empty, auto-generated by system.
</ParamField>

<ParamField path="reference_audio" type="Optional[str]" default="None">
  Path to a reference audio file for style transfer or cover tasks.
</ParamField>

<ParamField path="src_audio" type="Optional[str]" default="None">
  Path to a source audio file for audio-to-audio tasks.
</ParamField>

<ParamField path="audio_codes" type="str" default="''">
  Audio semantic codes as a string (advanced use, for code-control generation).
</ParamField>

<ParamField path="repainting_start" type="float" default="0.0">
  For repaint/lego tasks: start time in seconds for region to repaint.
</ParamField>

<ParamField path="repainting_end" type="float" default="-1">
  For repaint/lego tasks: end time in seconds for region to repaint (`-1` for until end).
</ParamField>

<ParamField path="chunk_mask_mode" type="str" default="'auto'">
  Chunk mask mode. Options: `"explicit"` = 0/1 mask from repaint range; `"auto"` = all 2.0 (model decides).
</ParamField>

<ParamField path="audio_cover_strength" type="float" default="1.0">
  Strength of reference audio/codes influence (range 0.0-1.0). Set smaller (0.2) for style transfer tasks.
</ParamField>

<ParamField path="cover_noise_strength" type="float" default="0.0">
  Noise strength for cover tasks. `0` = pure noise (no cover), `1` = closest to src audio.
</ParamField>

## Language Model Parameters

5Hz Language Model parameters for Chain-of-Thought reasoning.

<ParamField path="thinking" type="bool" default="True">
  If `True`, enable 5Hz Language Model "Chain-of-Thought" reasoning for semantic/music metadata and codes.
</ParamField>

<ParamField path="lm_temperature" type="float" default="0.85">
  Sampling temperature for the LLM (0.0-2.0). Higher = more creative/varied results.
</ParamField>

<ParamField path="lm_cfg_scale" type="float" default="2.0">
  Classifier-free guidance scale for the LLM.
</ParamField>

<ParamField path="lm_top_k" type="int" default="0">
  LLM top-k sampling (`0` = disabled).
</ParamField>

<ParamField path="lm_top_p" type="float" default="0.9">
  LLM top-p nucleus sampling (`1.0` = disabled).
</ParamField>

<ParamField path="lm_negative_prompt" type="str" default="'NO USER INPUT'">
  Negative prompt to use for LLM (for control).
</ParamField>

<ParamField path="use_cot_metas" type="bool" default="True">
  Whether to let LLM generate music metadata via CoT reasoning.
</ParamField>

<ParamField path="use_cot_caption" type="bool" default="True">
  Whether to let LLM rewrite or format the input caption via CoT reasoning.
</ParamField>

<ParamField path="use_cot_lyrics" type="bool" default="False">
  Whether to let LLM generate/format lyrics via CoT reasoning. (Not yet used)
</ParamField>

<ParamField path="use_cot_language" type="bool" default="True">
  Whether to let LLM detect vocal language via CoT.
</ParamField>

<ParamField path="use_constrained_decoding" type="bool" default="True">
  Whether to use FSM-based constrained decoding for the LLM.
</ParamField>

<ParamField path="cot_bpm" type="Optional[int]" default="None">
  CoT-generated BPM (output field, set by LLM).
</ParamField>

<ParamField path="cot_keyscale" type="str" default="''">
  CoT-generated key scale (output field, set by LLM).
</ParamField>

<ParamField path="cot_timesignature" type="str" default="''">
  CoT-generated time signature (output field, set by LLM).
</ParamField>

<ParamField path="cot_duration" type="Optional[float]" default="None">
  CoT-generated duration (output field, set by LLM).
</ParamField>

<ParamField path="cot_vocal_language" type="str" default="'unknown'">
  CoT-detected vocal language (output field, set by LLM).
</ParamField>

<ParamField path="cot_caption" type="str" default="''">
  CoT-enhanced caption (output field, set by LLM).
</ParamField>

<ParamField path="cot_lyrics" type="str" default="''">
  CoT-generated lyrics (output field, set by LLM).
</ParamField>

## Usage Example

```python
from acestep.inference import GenerationParams, GenerationConfig, generate_music

# Create parameters for a 30-second rock song
params = GenerationParams(
    caption="Energetic rock song with electric guitar riffs",
    lyrics="[Instrumental]",
    bpm=140,
    keyscale="E minor",
    duration=30,
    instrumental=True,
    inference_steps=8,
    seed=42,
    enable_normalization=True,
    normalization_db=-1.0,
    thinking=True,
    use_cot_metas=True
)

config = GenerationConfig(
    batch_size=2,
    use_random_seed=False
)

# Generate music
result = generate_music(
    dit_handler=handler,
    llm_handler=llm_handler,
    params=params,
    config=config,
    save_dir="./output"
)

if result.success:
    for audio in result.audios:
        print(f"Generated: {audio['path']}")
```
